===============================================
  虛擬幣價格查詢工具 - 一鍵啟動說明
===============================================

【Windows 用戶】
1. 確保已安裝 Python 3.7+（安裝時要勾選 "Add Python to PATH"）
2. 雙擊執行 start_gui.bat
3. 等待自動安裝依賴（第一次需要 1-2 分鐘）
4. GUI 會自動啟動

【macOS/Linux 用戶】
1. 確保已安裝 Python 3.7+
2. 雙擊執行 start_gui.sh 或在終端機執行：./start_gui.sh
3. 等待自動安裝依賴（第一次需要 1-2 分鐘）
4. GUI 會自動啟動
