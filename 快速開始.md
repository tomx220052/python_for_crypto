# 🚀 快速開始（給第一次使用的客戶）

## Windows 用戶

### 步驟 1：安裝 Python（一次性設定）

1. 點擊下載 Python：https://www.python.org/downloads/
2. 下載後執行安裝程式
3. **⚠️ 重要：勾選「Add Python to PATH」**
4. 點擊「Install Now」
5. 等待安裝完成

![Python 安裝示意](https://docs.python.org/3/_images/win_installer.png)

### 步驟 2：啟動工具

1. 找到專案資料夾中的 **`start_gui.bat`** 檔案
2. **雙擊執行** `start_gui.bat`
3. 第一次執行會自動安裝需要的套件（需要等待 1-2 分鐘）
4. 安裝完成後會自動開啟 GUI 視窗

### 以後使用

每次只要**雙擊 `start_gui.bat`** 就能直接開啟工具！

---

## macOS 用戶

### 步驟 1：安裝 Python（一次性設定）

#### 方法 1：使用 Homebrew（推薦）
1. 開啟「終端機」（Terminal）
2. 複製貼上以下指令並按 Enter：
   ```bash
   brew install python3
   ```

#### 方法 2：官方安裝包
1. 下載：https://www.python.org/downloads/macos/
2. 執行安裝程式

### 步驟 2：啟動工具

1. 找到專案資料夾中的 **`start_gui.sh`** 檔案
2. **雙擊執行** `start_gui.sh`
   - 如果無法執行，右鍵選擇「打開」
3. 第一次執行會自動安裝需要的套件
4. 安裝完成後會自動開啟 GUI 視窗

### 以後使用

每次只要**雙擊 `start_gui.sh`** 就能直接開啟工具！

## 📱 使用說明（開啟 GUI 後）

### 1. 輸入幣種 ID
在「幣種 ID」欄位輸入，例如：
- `bitcoin` - 比特幣
- `ethereum` - 以太坊
- `tether` - 泰達幣

💡 更多幣種請到 https://www.coingecko.com/ 查詢

### 2. 選擇日期
- **開始日期**：例如 `2025-11-01`
- **結束日期**：例如 `2025-11-10`
- 最多可查詢 100 天

### 3. API Key（可選）
- 如果有 CoinGecko API Key 可以輸入
- 沒有也可以直接查詢

### 4. 點擊「開始查詢」
等待查詢完成（會顯示進度）

### 5. 匯出結果
點擊「📥 匯出 CSV」儲存結果

## ❓ 遇到問題？

### Windows 常見問題

**Q: 雙擊 `start_gui.bat` 沒反應？**
- 確認是否已安裝 Python
- 重新安裝 Python 並確保勾選「Add Python to PATH」

**Q: 視窗一閃就關閉？**
- 對 `start_gui.bat` 點右鍵 → 「編輯」
- 最後一行加上 `pause`
- 儲存後再執行，就能看到錯誤訊息

**Q: 防毒軟體阻擋執行？**
- 將專案資料夾加入防毒軟體的白名單
- 或暫時關閉防毒軟體

### macOS 常見問題

**Q: 雙擊 `start_gui.sh` 沒反應？**
- 右鍵點擊 `start_gui.sh` → 選擇「打開」
- 或在終端機執行：`chmod +x start_gui.sh && ./start_gui.sh`

**Q: 顯示「無法驗證開發者」？**
- 到「系統偏好設定」→「安全性與隱私」
- 點擊「強制打開」
